# Changelog

## 0.1.0

Initial release.

- Filter short-lived opposite-direction wheel events.
- Emit a virtual mouse through `uinput`.
- Add configurable `--lock-ms` debounce window.
- Add configurable `--scroll-mult` wheel speed multiplier.
- Add device listing, debug mode, and dry-run mode.
- Add Fedora/Wayland troubleshooting notes and systemd examples.
